◆ MILLI_UNIT_PER_LSB_TO_UNIT

#define MILLI_UNIT_PER_LSB_TO_UNIT   2.0f / 65535.0f

All units being converted are being stored in milli-units.

Ex: Milli-Gs per least significant bit to Gs conversion rate using 32g full scale range. 32g FSR * 2 = 64g range * 1000 = 64000mg range Result is given in 16b integer, so 65535 possible different values for 64000mg Conversion factor = 64000/65535 (mg per bit) See https://ozzmaker.com/accelerometer-to-g/ We will return units in regular units instead of milli-units, so the multiplication of 1000 is removed.
