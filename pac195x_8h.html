int pac195x_toggle_channel ( SensorLocation const *  loc,
pac195x_channel_e  channel,
bool  enable 
)

Enable/disable channels for sampling on the PAC195X. NOTE: Some models, like the PAC1952-2, physically do not have all channel pins. Trying to enable those channels will do nothing.

Parameters
locThe location of the sensor on the I2C bus.
channelA channel or multiple channels to enable/disable. Multiple channels can be given by ORing the channels together.
enableTrue to enable the channel(s), false to disable the channel(s).
Returns
Any error which occurred while communicating with the sensor. EOK if successful.
