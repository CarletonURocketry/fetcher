int m24c0x_write_page ( SensorLocation const *  loc,
uint8_t  addr,
uint8_t const *  data,
size_t  nbytes 
)

Write a page (16 bytes) of data to the EEPROM starting at addr. WARNING: If the data is too long to fit in the page, the bytes exceeding the page end are written on the same page from location 0.

Parameters
locThe location of the EEPROM on the I2C bus.
addrThe address of the EEPROM to write the data to.
dataA buffer with data to write to the EEPROM.
nbytesThe number of bytes in the data buffer to be written to the EEPROM.
Returns
0 if successful, the error that occurred otherwise.
