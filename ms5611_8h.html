Parameters
locThe location of the MS5611 sensor on the I2C bus.
resThe resolution to read the MS5611 at.
ctxThe context containing calibration coefficients and ground pressure.
preciseTrue to use second order calculation for higher precision, false for quicker calculation.

altitudeStorage location of the altitude value in m. NULL to skip calculation.
