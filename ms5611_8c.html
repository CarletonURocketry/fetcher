Detailed Description

Sensor API interface implementation for the MS5611 pressure and temperature sensor.

Sensor API interface implementation for the MS5611 pressure and temperature sensor which uses I2C communication.

See https://www.te.com/commerce/DocumentDelivery/DDEController?Action=showdoc&DocId=Data+Sheet%7FMS5611-01BA03%7FB3%7Fpdf%7FEnglish%7FENG_DS_MS5611-01BA03_B3.pdf%7FCAT-BLPS0036. for the MS5611 data sheet:

See https://www.amsys-sensor.com/downloads/notes/ms5611-precise-altitude-measurement-with-a-pressure-sensor-module-amsys-509e.pdf for information about the MS5611 altitude measurement calculations.

Parameters

preciseTrue to use second order calculation for higher precision, false for quicker calculation.
