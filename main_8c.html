uint8_t *  naddrs

Reads up to naddrs addresses from the board ID into an array of addresses.

Parameters
board_idThe current position in the board ID where an address starts.
addressesAn array of bytes with enough space to store naddrs bytes.
naddrsA pointer to the maximum number of addresses to read. After the function call, this pointer will contain the actual number of addresses read.
Returns
The current position in the board ID after the read, or NULL if naddrs needs to be larger.
