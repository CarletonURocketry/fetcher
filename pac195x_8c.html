Detailed Description

Includes a driver for the PAC195X family of power monitors. Currently tested on the PAC1952. Datasheet: https://ww1.microchip.com/downloads/aemDocuments/documents/MSLD/ProductDocuments/DataSheets/PAC195X-Family-Data-Sheet-DS20006539.pdf.

Enumeration Type Documentation

◆ pac195x_reg_t

enum pac195x_reg_t

All the different registers/commands available in the PAC195X.

Enumerator
REFRESH 

Refreshes the PAC195X.

CTRL 

Control register for configuring sampling mode and ALERT pins.

ACC_COUNT 

Accumulator count for all channels.

VACCN 

Accumulator outputs for channels 1-4. Spans until 0x06.

VBUSN 

V_BUS measurements for channels 1-4. Spans until 0x0A.

VSENSEN 

V_SENSE measurements for channels 1-4. Spans until 0x0E.

VBUSN_AVG 

Rolling average of the 8 most recent V_BUS_n measurements, (n: 1-4). Spans until 0x12.

VSENSEN_AVG 

Rolling average of the 8 most recent V_SENSE_n measurements, (n: 1-4). Spans until 0x16.

VPOWERN 

V_SENSE * V_BUS for channels 1-4. Spans until 0x1A.

SMBUS_SETTINGS 

Activate SMBus functionality, I/O data for R/W on I/O pins.

NEG_PWR_FSR 

Configuration control for bidirectional current.

REFRESH_G 

Refresh response to General Call Adddress.

REFRESH_V 

Refreshes V_BUS and V_SENSE data only, no accumulator reset.

SLOW 

Status and control for SLOW pin functions.

CTRL_ACT 

Currently active value of CTRL register.

NEG_PWR_FSR_ACT 

Currently active value of NEG_PWR register.

CTRL_LAT 

Latched active value of CTRL register.

NWG_PWR_FSR_LAT 

Latched active value of NEG_PWR register.

ACCUM_CONFIG 

Enable V_SENSE and V_BUS accumulation.

ALERT_STATUS 

Reads to see what triggered ALERT.

SLOW_ALERT1 

Assigns specific ALERT to ALERTn/SLOW.

GPIO_ALERT2 

Assigns specific ALERT to ALERTn/I/O.

ACC_FULLNESS_LIMITS 

ACC and ACC Count fullness limits.

OC_LIMITN 

OC limit for channels 1-4. Spans until 0x33.

UC_LIMITN 

UC limit for channels 1-4. Spans until 0x37.

OP_LIMITN 

OP limit for channels 1-4. Spans until 0x3B.

OV_LIMITN 

OV limit for channels 1-4. Spans until 0x3F.

UV_LIMITN 

UV limit for channels 1-4. Spans until 0x43.

OC_LIMIT_NSAMPLES 

Consecutive OC samples over threshold for ALERT.

UC_LIMIT_NSAMPLES 

Consecutive UC samples over threshold for ALERT.

OP_LIMIT_NSAMPLES 

Consecutive OP samples over threshold for ALERT.

OV_LIMIT_NSAMPLES 

Consecutive OV samples over threshold for ALERT.

UV_LIMIT_NSAMPLES 

Consecutive UV samples over threshold for ALERT.

ALERT_ENABLE 

ALERT enable.

ACCUM_CONFIG_ACT 

Currently active value of ACCUM_CONFIG register.

ACCUM_CONFIG_LAT 

Currently latched value of ACCUM_CONFIG register.

PRODUCT_ID 

The register containing the product ID.

MANUFACTURER_ID 

The register containing the manufacturer ID of 0x54.

REVISION_ID 

The register containing the revision ID. Initial release is 0x02.
