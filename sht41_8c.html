int sht41_heat ( SensorLocation const *  loc,
sht41_dur_e  duration,
sht41_wattage_e  wattage,
float *  temperature,
float *  humidity 
)

Heats up the SHT41 sensor and then performs a high precision read of temperature and humidity. WARNING: May draw a large amount of current.

Parameters
locThe location of the SHT41 sensor on the I2C bus.
durationThe duration to heat the SHT41 sensor for.
wattageThe wattage to use when heating the SHT41 sensor.
temperatureA pointer to store the temperature in degrees Celsius.
humidityA pointer to store the relative humidity in percentage.
Returns
EOK if successful, otherwise returns the error that occurred.
